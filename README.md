# 2D-Vector-Drawer

- 2D Vector Drawer with Adding, Subtracting, and Matrix Multiplication, using PyQt6 and pyqtgraph

### Features

- Python script that graphs 2d vectors and can add or subtract 2 vectors and graphs the result and has the ability to calculate linear transformations and graphs the result

### Dependencies
- PyQT6
- PIL
- pyqtgraph
- Numpy
